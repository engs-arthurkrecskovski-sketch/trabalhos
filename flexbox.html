O que é o Flexbox?

O Flexbox (ou Flexible Box Layout) é um modelo de layout do CSS que facilita o alinhamento e a distribuição de espaço entre os elementos dentro de um contêiner, mesmo quando o tamanho dos itens é desconhecido ou dinâmico.

Propriedades principais

  • display: flex; — ativa o flexbox no contêiner.
  • justify-content — alinha os itens no eixo principal (horizontal).
  • align-items — alinha os itens no eixo cruzado (vertical).
  • flex-direction — define a direção dos itens (row, column, etc).
